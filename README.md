# 📊 Walmart Sales & Profit Dashboard (Excel)

## 📌 Overview

This project is an interactive Excel dashboard created to analyze sales and profit performance.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Slicers
* Charts

## 📊 Key Insights

* Monthly sales trends

* City-wise analysis
* Payment method insights
